關於 SITCON 學生計算機年會

學生計算機年會（Students' Information Technology Conference）自 2012 年發起，以學生為本、由學生自發舉辦，長期投身學生資訊教育與推廣開源精神，希望引領更多學子踏入資訊的殿堂，更冀望所有對資訊有興趣的學生，能夠在年會裏齊聚一堂，彼此激盪、傳承、啟發，達到「學以致用、教學相長」的實際展現。

關於 Hour of Code

Hour of Code（一小時玩程式） 是一項全球推廣計畫，由非營利組織 Code.org 發起，旨在讓每個人都能在短時間內體驗寫程式的樂趣與可能性。透過簡單、有趣的互動課程，參與者只需花上短短的時間，就能初步理解程式設計的邏輯思維，發現運算思維在日常生活中的應用，並開啟通往科技世界的第一步。

桃園場

本場次教學內容為 基礎程式邏輯思維入門。課程以淺顯易懂的方式帶領學員認識程式設計的基本概念，透過遊戲化的練習與互動活動，培養邏輯思考與問題解決能力。學員將在輕鬆的氛圍中體驗「寫程式」的核心精神，為未來的進階學習打下基礎。

宜蘭場

本場次教學內容為 mBot 教育機器人編程體驗。透過圖像化積木程式設計，學員將學習控制機器人前進、轉彎與感測環境，從中培養邏輯思考與問題解決能力。課程以實作為主，讓孩子在動手操作中理解程式設計的原理，啟發創造力與對科技的興趣。

高雄場

本場次教學內容為資訊安全與社交工程認識。課程將以「陌生人的糖果」作為比喻，說明社交工程的危險性，並透過互動式網頁實作，模擬社交工程的常見手法。學員將在體驗過程中理解資訊安全的重要性，培養辨識與防範社交工程的能力，提升面對網路世界的安全意識。

Hour of Code 是 SITCON 推廣教育活動的一環，讓更多人有機會接觸電腦科學、理解開源社群文化。
